# 地球板块移动历史 - 3D可视化

![GitHub](https://img.shields.io/badge/版本-1.0.0-blue)
![GitHub](https://img.shields.io/badge/Three.js-r128-green)

一个交互式的3D地球板块移动历史可视化网站，展示从现代到史前时代的地球板块变化。

## 🌍 在线演示

访问网站：https://[您的用户名].github.io/earth-plate-tectonics

## ✨ 功能特性

- *3D地球模型* - 使用Three.js创建的可旋转和缩放的地球
- *时间轴控制* - 从现代(0百万年前)到寒武纪(5.42亿年前)
- *历史地貌展示* - 不同年代的地球板块分布
- *交互式控制* - 暂停旋转、重置视角
- *星空背景* - 增强视觉体验的星空效果
- *响应式设计* - 支持桌面和移动设备

## 🛠️ 技术栈

- *前端*: HTML5, CSS3, JavaScript (ES6+)
- *3D图形库*: [Three.js](https://threejs.org/) r128
- *部署平台*: GitHub Pages
- *版本控制*: Git
